# Learning Playwright

This repository contains learning materials and examples for Playwright, a modern end-to-end testing framework for web applications.

## Project Structure

```
learning-playwright/
├── Capter_01_Basics/          # Chapter 1: Basic Playwright concepts
│   ├── Basics.js              # Basic Playwright setup and examples
│   ├── Step_by_step.js        # Step-by-step guide to Playwright
│   └── Verify_setup.js        # Setup verification script
└── chapter_02_javaprograming/  # Chapter 2: Java Programming concepts
```

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Playwright browsers (auto-installed)

### Installation

```bash
# Clone the repository
git clone https://github.com/Dwijeshb/learning-playwright.git
cd learning-playwright

# Install dependencies
npm install

# Install Playwright browsers
npx playwright install
```

## Running Tests/Examples

```bash
# Run basic examples
node Capter_01_Basics/Basics.js

# Run step-by-step guide
node Capter_01_Basics/Step_by_step.js

# Verify your Playwright setup
node Capter_01_Basics/Verify_setup.js
```
